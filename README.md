# covid-19-ml-analysis-indonesia
Machine learning analysis of COVID-19 spread in Indonesia using clustering and logistic regression.
# Machine Learning Analysis of COVID-19 Spread in Indonesia

This project analyzes COVID-19 spread patterns across Indonesian regions using machine learning techniques.

## Methods
- K-Means Clustering for regional grouping
- Logistic Regression for case prediction
- Data preprocessing and feature engineering

## Tools
- Python
- Pandas
- Scikit-learn
- Streamlit

## Results
The logistic regression model achieved an ROC-AUC score of **0.88**.
